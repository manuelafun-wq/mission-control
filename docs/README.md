# Mission Control

Mission Control is an Executive Office.

Not a task manager. Not a productivity app. Not a dashboard.

An Executive Office is a place where a leader arrives, gets oriented, makes decisions, and moves. It is designed around the person at the top — not the tools they use.

---

## What it does

Mission Control gives Manuela one place to:

- Receive a daily briefing from Noah
- Decide what matters today
- Record ideas and decisions
- Track the projects that carry her vision forward

Everything else is noise. Mission Control removes it.

---

## What it is not

- Not a to-do list
- Not a project management tool
- Not a calendar
- Not a communication platform
- Not another app to maintain

---

## The premise

Most productivity software is built for employees. It optimizes for throughput, compliance, and visibility upward.

Mission Control is built for a Visionary. It optimizes for clarity, focus, and momentum forward.

The difference is everything.

---

## Stack

- Next.js (App Router)
- React
- Tailwind CSS v4
- localStorage (local memory, Phase 1)
- Claude API (Noah intelligence, Phase 3+)
